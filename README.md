# 🚀 ApplyFlow – Job Application Tracking System

👩‍💻 Developed by Anusha Banothu

A modern, feature-rich web application built using React and TypeScript to help users efficiently manage and track job applications. ApplyFlow provides powerful tools like filtering, analytics, notes management, and customizable fields — all in a clean and responsive UI.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## ✨ Features

### 📊 Dashboard & Application Management
- Comprehensive table view displaying all applications  
- Smart search across all fields for quick filtering  
- Status-based filtering with color-coded badges  
- Pagination support for better data handling  
- Edit, delete, and view application details  

### ✏️ Application & Notes Management
- Add/Edit applications with dynamic forms  
- Attach timestamped notes to each application  
- Form validation with error handling  
- Auto-save using browser local storage  

### 📈 Statistics & Analytics
- Track total number of applications  
- Visualize application status distribution  
- Analyze applications over time  
- Calculate response rate and trends  

### 🎨 Customization & UI
- Fully customizable fields  
- Dark/light mode support  
- Clean and responsive design  

---

## 🛠️ Tech Stack

- Frontend: React, TypeScript  
- State Management: Redux Toolkit  
- Styling: SCSS  
- Charts: Recharts  
- Build Tool: Vite

## 🎯 Key Highlights

- Focused on clean UI/UX and responsive design
